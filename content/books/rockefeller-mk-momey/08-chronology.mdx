# 洛克菲勒生平大事記 (Chronology)

這張表記錄了約翰·D·洛克菲勒（John D. Rockefeller, 1839-1937）從一名簿記員成長為全球首富，最後轉型為大慈善家的傳奇一生。

| 年份 | 事件 (Event) | 意義 (Significance) |
| :--- | :--- | :--- |
| **1839** | 出生於紐約州里奇福德 | 平凡的起點。 |
| **1855** | 在克里夫蘭擔任助理簿記員 | 9月26日被他定為「工作日」，意義超過生日。 |
| **1863** | 投資第一座煉油廠 | 正式踏入石油產業。 |
| **1870** | **創立標準石油公司 (Standard Oil)** | 開啟了全球能源壟斷的時代。 |
| **1882** | 組建標準石油信託 | 現代企業托拉斯結構的鼻祖。 |
| **1897** | 從公司日常管理中退休 | 開始將精力轉向慈善事業。 |
| **1911** | 美國最高法院裁定標準石油解散 | 分拆成 34 家公司（如現在的 ExxonMobil、Chevron）。 |
| **1913** | **成立洛克菲勒基金會** | 致力於醫療、教育與科學研究。 |
| **1937** | 以 97 歲高齡逝世 | 留下了龐大的商業遺產與慈善規範。 |

---

## 歷史影響

1.  **商業規範**：他定義了現代企業的組織結構與效率標準。
2.  **醫學貢獻**：洛克菲勒基金會資助了消滅黃熱病、鉤蟲病的研究，並參與創立了北京協和醫學院。
3.  **財富觀念**：他將「賺錢」與「捐錢」都視為一門科學，影響了後來的巴菲特與比爾蓋茲。
